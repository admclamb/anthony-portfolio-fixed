We Love Movies

V1.0.0

We Love Movies is a backend application that gathers movie data, critic data, theater data, and reviews data for movies. This application was made in the Thinkful Engineer Program and has taught me a lot in backend development. Before this application, I had very little backend development experience. This application uses express to create the server, and routes and controllers to handle CRUD with a RESTful API style. This application also uses postgreSQL and knex to save update and retrieve data.
