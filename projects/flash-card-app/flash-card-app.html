Card-O-Matic

V1.0.4

Card-O-Matic is a flash card application similiar to QuizUp where users can study there flashcards. This application was made with front end technologies like React, JavaScript, and Bootstrap 4. This flash card application also has a mock backend to save information. This application was made in the Thinkful Engineering Flex boot camp and I have improved upon the application after the boot camp. This project is still in development and I will be adding more features soon.
