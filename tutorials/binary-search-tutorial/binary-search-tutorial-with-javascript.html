Creating a binary search with JavaScript

May, 5, 2022
Author:
Anthony Mclamb

In this article we'll be building a binary search algorithm that'll first be able to search a normal array. Then afterwards, we'll show how to implement the algorithm into a tree data structure.

First well create a Search class with a static binary method so we will not have to instanciate a Search class to use the method.

                
    class Search {
        static binary(compare, sortedElements) {
        }
    }
                            
    module.exports = Search

The static binary method on the Search class will take two arguments, compare, sortedElements. 

                
    class Search {
        /*
          params: 
          compare is of type function and provides comparison of elements
          sortedElements is an array of elements in sorted form
        */
        static binary(compare, sortedElements) {

            // check if the elements is of type array

            if (Array.isArray(sortedElements)) {

                // initialize a lower index to the lowest possible index

                let lowerIndex = 0;

                // initialize a higher index to the highest possible value

                let higherIndex = sortedElements.length - 1;
                  
                // iterate until the lower index is higher than the higher index which
                // means there is no instance of the target

                while (lowerIndex <= higherIndex) {

                    // initialize the middle index by calculating from lower 
                    // and higher index

                    const middleIndex = Math.floor((higherIndex + lowerIndex) / 2);
                  
                    // compare the current middle index to the target inside of the
                    // compare function

                    const comparison = compare(sortedElements[middleIndex]);
                  
                    // if the current element is lower than the target

                    if (comparison > 0) {
                        lowerIndex = middleIndex + 1;
                    }

                    // if the current element is equal to the target

                    if (comparison === 0) {
                        return middleIndex;
                    }

                    // if the current element is higher than the target

                    if (comparison < 0) {
                        higherIndex = middleIndex - 1;
                    }
                }
            }

            // if there is no target in the element return -1

            return -1;
        }
    }
                  
                  
    module.exports = Search

Next we will make a comparison function that takes the target as an argument and returns an anonymous function that'll take an element as an argument that we will be used to compare. When this compare function is called in the binary search it will pass the arguments to the anonymous function thats being returned. This will give us access to the target and keep us from hard coding our comparison so we can add other comparisons later.

Lastly we'll test out our code by created an array of sorted elements and a target then console.log() our index.
