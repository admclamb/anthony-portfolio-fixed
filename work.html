Devify

Devify is a social media application that uses modern technologies such as the PERN stack. This application is based from the Dev.to platform and allows users to create and interact with each other. Devify uses React Hooks and React State to manage users data and Node.js with PostgreSQL and AWS S3 Buckets to handle the data. This application uses modern practices to store users information by using hasing data and JWT to create user sessions.

Weatherly

Weatherly is a responsive website MOCK application that gives real world and real time weather data. This application uses OpenWeather API, GeoLocation API, and NewsData.io API to get a users location and conditionally display the features. Weatherly uses React Hooks and React state to manage all the data going through the application and renders the app depending on location, weather, etc.
